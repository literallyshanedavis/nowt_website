# Design

## Theme

Dark, cinematic, and image-led. The physical scene is a creative person using an iPad in low ambient light on location, checking a workflow between moments rather than sitting at a desk.

## Color

Use tinted near-black surfaces instead of pure black. The core palette is restrained: warm off-white text, smoky charcoal panels, muted teal for actions, and occasional warm red/orange media light.

## Typography

Use the existing Geist family for the brand surface. Keep headings large, light, and calm. Use Geist Mono only for short navigation labels, node labels, pills, and technical captions.

## Layout

Follow a black-stage, cinematic production-site rhythm: fixed sparse nav, full-viewport video hero, large lower-left headline overlapping the bottom of the hero media, generous black intro bands, three editorial media columns, wide feature films, a horizontal filmstrip, and a simple closing waitlist.

## Components

Navigation should be sparse and fixed. Buttons are rounded pills with restrained fills. Media cards use low radius and real-feeling visual treatments. Avoid nested cards, repeated icon grids, and node-diagram hero treatments.

## Motion

Use subtle slow movement: filmstrip drift, media glow shifts, node pulses, and reveal-friendly static composition. Respect `prefers-reduced-motion`.
